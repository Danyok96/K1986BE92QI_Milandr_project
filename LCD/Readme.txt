Для работы с дисплеем в main файл проекта необходимо прописать:
#include "lcd.h"
#include "text.h"
#include "font_defs.h"

В проект нужно добавить файлы (файлы физически должны располагаться в папке проекта):
Font_6x8.c
lcd.c
systick.c
text.c

Для работы с дисплеем используются следующие функции:

LCD_INIT(); - инициализация дисплея, вызывается после настройки частоты микроконтроллера.

LCD_CLS(); - очистка дисплея.

LCD_PUTS(x, y, *str); - вывод строки на дисплей. В качестве параметров: x - смещение текста по горизонтали (слева на право), y - смещение текста по вертикали (сверху вниз), *str - указатель на выводимую строку.

Перед вызовом LCD_PUTS(x, y, *str) необходимо присвоить значения двум переменным (достаточно сделать один раз, после инициализации дисплея): 
CurrentFont = &Font_6x8; 
CurrentMethod = MET_AND;
